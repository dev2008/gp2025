How It Works

  • Field values: To display a field from your database, wrap the field name between two dadabik_field tags.
    Example: dadabik_field title_book dadabik_field

  • Permissions: These links do not respect user permissions automatically. If you want to hide links from specific user groups, wrap them in dadabik_hide and dadabik_groups tags. In this example, the EDIT and DELETE links are hidden for groups 2 and 3.

  • Styling: You can use any HTML to style the layout (e.g., <div>, <span>, inline CSS) to create a custom layout that matches your application's design.

  • Do not use reserved keywords: Make sure that your data (field content) does not contain any of the following:
    • dadabik_field
    • dadabik_link
    • dadabik_hide
    • dadabik_groups
    These are reserved and will break the template if found in field values.
